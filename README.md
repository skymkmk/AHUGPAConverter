# 安大教务四分制绩点转换

***从 1.3 版本开始脚本更新了命名空间，会导致脚本重复，请将原来的脚本删除再安装本脚本。1.3 之后的版本无需进行该操作！***

&emsp;&emsp;深夜睡不着.jpg，于是瞎写了个很简单的小脚本。方便大家计算自己的四分制成绩，方便开润。

&emsp;&emsp;使用方法：教务系统->信息查询->成绩查询->查询已修课程最高成绩。不出意料的话就会在页面上显示你的四分制成绩。（采用北大算法和 WES 算法计算）
